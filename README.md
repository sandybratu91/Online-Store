# Online-Store
Data filtering from an online store

Task objective:
Develop a Python script that allows initial analysis and targeting in a
data set containing information about the products of an online store.
Emphasis is on using the library's core functionality
Pandas to load, view, filter and sort data.

How many products are there in the dataset?
Display the total number of rows in the DataFrame.
What is the best selling product in the entire store
online?
Sort the DataFrame by the number of units sold and
display the first row.
What are the top 5 best selling mobile phones?
Filter products from the "Smartphones" category, then sort-
them by the number of units sold and select the top 5.
What is the price of the most expensive and cheapest laptop
Filter products from the "Laptops" category, remove
rows with missing values ​​and find the maximum and minimum price.
